st - simple terminal Version:0.8.5
---
 st is a simple terminal emulator for X which sucks less.
 
 

Keys
---
 This build of st is set to-


Mod Key
---

    Default Mod: Super (Mod4)    
    TermMod:     Ctrl+Shift
   
Actions
---

    Copy:        Ctrl+Shift+c
    Paste:       Ctrl+Shift+v
    
    Color:       Ctrl+Shift+p
    
    Zoom:
      - In:      Ctrl+Shift+pgup
      - Out:     Ctrl+Shift+pgdn
      - Reset:   Ctrl+Shift+home
      
    Scroll:
      - Up:      Alt+Ctrl+k
      - Down:    Alt+Ctrl+j
    
    Url:
      - Select:  Super+l
      - Open:    Super+u
